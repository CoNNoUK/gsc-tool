## GSC Tool
A utility to manage IW engine game scripts, made of 3 modules (Assembler, Disassembler & Decompiler) and a command line app.

 Supported games: IW5
## Usage
``./gsc-tool.exe <options> <file>``

available options:
* ``-asm``
* ``-disasm``
* ``-decomp``
